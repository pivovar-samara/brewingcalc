# Contributing to BrewingCalc

## Code Style

- Swift 6 with strict concurrency
- SwiftUI for all UI code
- Value types (structs) preferred over classes
- No force unwraps or force casts
- Use `String(localized:)` for all user-facing text

## Pull Requests

1. Fork the repository
2. Create a feature branch from `main`
3. Write tests for new functionality
4. Ensure all tests pass
5. Submit a pull request with a clear description

## Reporting Issues

Open an issue with:
- Steps to reproduce
- Expected vs actual behavior
- iOS version and device type

## Calculation Formulas

The brewing calculation formulas are well-established and should not be modified without strong justification and updated test cases. If proposing formula changes, include references to the source material.
