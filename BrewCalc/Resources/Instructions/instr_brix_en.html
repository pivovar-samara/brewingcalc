Refractometer calculator

Allows to calculate the parameters of beer from refractometer measurements - Brix. Has 2 modes:
  1. Brix. If you know original and final gravity in Brix scale you can calculate this values in Plato or hydrometer scale, as well as alcohol by volume value (ABV).
  2. Final measurements. Use this mode in case of you have the final values in Brix and hydrometer scales and want to calculate original gravity and ABV value.

Correction factor

This factor can be different for different breweries.
How to calculate it:
  1. Measure the specific gravity with refractometer.
  2. Measure the specific gravity with hydrometer.
  3. Convert hydrometer value to Brix using the equation Brix=(SG-1)/0.004, where SG - specific gravity measured by hydrometer.
  4. Devide refractometer value (1) by actual hydrometer value (3). Now you have a number between 1.02 and 1.06.
  5. Repeat this calculations several times and calculate the average value. You can use this number as a correction factor for your brewery. The factor is about 1.04 in general and we use this value by default.
